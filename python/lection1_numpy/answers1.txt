1) Функция numpy.arrange(start, end, step) создаёт последовательность от start до stop, при этом end в неё не включается.
Если step не указан, то шаг = 1.
Функция numpy.linspace(start, end, size) создаёт последовательнсоть из size чисел с одинаковым интервалом в диапазоне (start, end)

2) Срез массива.
Иммет слудющую конструкцию: arr[start : end : step]. По этому правилу выбирается подмассив элементов массива arr, начиная с индекса
start и заканичая индексом end-1 с шагом step.

3) m - двумерный массив
m[:, j] - все элементы столбца j (одномерный массив)
m[i, :] - все элементы строки i (одномерный массив)

4) Отрицательный индексы указывают на то, что отсчёт идёт с конца массива.

5) np.argmax(arr) - возвращается индекс максимального элемента массива
np.argmin(arr) - возвращается индекс минимального элемента массива

6) Универсальные функции - функции, выполняющие обработку массивов типа ndarray поэленментно.
Например a, b - массивы.
a + b
a - b
a * b
a / b
